# <span style={{color: '#4fddbf'}}>Java 시작하기</span>

해당 문서는 ‘자바의 정석’ 책을 토대로 이외에 구글링, 여러 문서를 통해 습득한 Java관련 지식을 정리한 내용입니다.

### 잊고 있었거나 새로운 깨달음

- Java API문서에서 모르는 라이브러리들을 찾아보자. 공식 문서이기 때문에 틀린 정보가 없을 것이며 코딩 테스트에서 구글링이 불가능한 시험의 경우에도 공식문서는 제공해주는 경우가 많기 때문에 도움이 될 것이다.
- .Java →(javac.exe(컴파일러)) → .class (Byte Code)→ (java.exe(인터프리터)) + Jit Compiler → 실행 결과 (Native Code)
